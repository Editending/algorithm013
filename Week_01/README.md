## 二、刻意练习<br>

* 最大误区：只刷一遍
* 五毒神掌：
	* 第一遍：
		* 5分钟读题并思考，最多15分钟
			* 有思路->开写
			* 没思路->直接看解法，注意！！多解法，比较优劣
		* 背诵默写
	* 第二遍：马上自己默写并提交，多种解法比较体会
	* 第三遍：一天后重复
	* 第四遍：一周后重复
	* 第五遍：面试前一到两周恢复性训练
